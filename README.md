## Swing analysis

A rec-league baseball swing-analysis checklist — watch a swing video, score 6
common "swing-killers," and hand the hitter 1–3 things to work on.

- **[Printable scorecard (PDF)](swing-analysis/scorecard.pdf)** — print blank
  copies to mark by hand, or text/email it to teammates. No app needed.
- **[Fill-in worksheet (HTML)](swing-analysis/scorecard.html)** — open in any
  browser to fill on screen or print to PDF.
- **[Full write-up](swing-analysis/swing-scorecard.md)** — the source checklist
  with the camera angles, cues, and drills explained.

The PDF is regenerated from `swing-analysis/build-scorecard-pdf.py`
(`pip install reportlab && python3 build-scorecard-pdf.py`).
